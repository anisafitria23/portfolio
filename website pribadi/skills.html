Skills

Berikut adalah beberapa kemampuan yang saya miliki dan terus saya kembangkan melalui pembelajaran serta pengerjaan tugas dan proyek.

HTML

Saya mampu membangun struktur halaman website menggunakan HTML dengan rapi dan terorganisir. Saya memahami penggunaan elemen semantik serta pembuatan form dasar untuk menghasilkan struktur halaman yang jelas dan mudah dipahami.

CSS

Saya dapat melakukan styling website menggunakan CSS untuk menciptakan tampilan yang konsisten dan responsif. Saya terbiasa mengatur layout dengan flexbox, mengelola warna dan tipografi, serta menyesuaikan tampilan di berbagai ukuran layar.

Creative Tools

Saya memiliki pengalaman mengedit foto dan video menggunakan Canva, CapCut, dan Lightroom. Kemampuan ini saya gunakan untuk membuat konten visual yang sederhana, rapi, dan menarik untuk kebutuhan presentasi maupun media digital.

Public Speaking

Saya memiliki kemampuan public speaking sebagai pembawa acara (MC) menggunakan Bahasa Indonesia dan Bahasa Jawa. Saya terbiasa berbicara di depan umum dengan bahasa yang sopan, jelas, dan komunikatif.
